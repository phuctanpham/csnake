## A/ Setup to play
1) Chạy lệnh sau để tạo file snake

```
clang++ main.cpp snake.cpp labyrinth.cpp -o snake

```
2) Chạy file snake vừa tạo

```
./snake

```
3) Chọn chế độ chơi và bắt đầu chơi
